## A1. 搜索引擎检索语法

| 作用       | 语法                                                       | 注释                                                                        |
| -------- | -------------------------------------------------------- | ------------------------------------------------------------------------- |
| 一般匹配     | `$KEYWORD`                                               |                                                                           |
| 精确匹配     | `"$KEYWORD"`                                             |                                                                           |
| 模糊匹配     | `K*WORD`                                                 | 通配符 wildcard                                                              |
| 限定网域     | `site:$URL`                                              |                                                                           |
| 限定文件类型   | `filetype:$FILETYPE`                                     |                                                                           |
| 限定标题     | `intitle:$KEYWORD` OR  <br>  <br>`allintitle: $KEYWORD`  |                                                                           |
| 限定URL    | `inurl: $KEYWORD` OR  <br>  <br>`allinurl: $KEYWORD`     | `inurl: gov.cn`  <br>  <br>`inurl: edu.cn`  <br>  <br>`inurl: moe.edu.cn` |
| 关键词定义    | `define:$KEYWORD`                                        |                                                                           |
| 布尔逻辑 NOT | `-$KEYWORD`                                              | 排除关键词                                                                     |
| 布尔逻辑 AND | `$KEYWORD1 $KEYWORD2`                                    |                                                                           |
| 布尔逻辑 OR  | `$KEYWORD OR $KEYWORD`  <br>  <br>`$KEYWORD \| $KEYWORD` |                                                                           |
| 必须包含     | `+$KEYWORD`                                              |                                                                           |
